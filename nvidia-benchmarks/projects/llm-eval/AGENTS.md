# AGENTS.md - LLM Evaluation Project

## Context
- **Goal:** Benchmarking and trace analysis for LLM systems.
- **Focus:** RAG evaluation, embeddings, and "LLM-as-a-judge" patterns.

## Tasks
- Always check `STATUS.md` before starting work.
- Consult `DATA_DICTIONARY.md` for schema/metric definitions.
- Log every experiment run in the `log/` directory with a date prefix.
- Document architectural choices in `docs/decisions/`.
- Use `exa` for research on benchmarks.
- Output evaluation results to the `results/` folder.
